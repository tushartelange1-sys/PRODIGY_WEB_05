Weather Web App
================

What it is
---------
A simple web page that fetches current weather from OpenWeatherMap and displays:
- Location
- Temperature (Â°C)
- Feels like
- Humidity
- Wind speed
- Sunrise and sunset times

How to use
----------
1. Get an API key from OpenWeatherMap: https://openweathermap.org/appid
2. Open `app.js` and replace `YOUR_API_KEY` with your API key.
3. Open `index.html` in your browser.
4. Click 'Use my location' (allow browser to share location) or search by city.
